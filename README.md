## Наш первый новостной сайт с группой 01-25

### Инструкции по установке:
### скачиваем этот проект через команду 
#### git clone https://github.com/edzen12/djangoFirst-01-25.git
### открываем в VSCode, открываем терминал
### и пишем команду 
#### python -m venv venv
#### .\venv\Scripts\activate
#### pip install django pillow django-ckeditor-5
#### python manage.py migrate
#### python manage.py createsuperuser
#### python manage.py runserver
